
# Évaluation meubles

Application de présentation de meubles pour notre client B. Ricci (avec statistiques, et possibilité d'ajout de meubles). Réalisée dans le cadre de la troisième semaine de Node.JS

## Technologies 

- React
- React Router
- Typescript
- Express
- Tailwind
- JWT
- MongoDB
